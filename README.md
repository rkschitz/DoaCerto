# TCS-DoaCerto

## 📦 Clonando o Repositório

Clone o repositório com o seguinte comando:

```bash
git clone git@github.com:rkschitz/DoaCerto.git
```

## 🚀 Executando o Frontend e Backend

Após clonar o repositório, acesse as pastas frontend e backend separadamente via terminal e execute:
```bash
npm install && npm start
```

#### Pronto, o projeto ja estará rodando
